## 📱 Responsividade

### Desktop (> 1024px)
- Layout completo com Sidebar visível
- Grid de 3-4 colunas
- Hero section expandido
- Todas as features visíveis

### Tablet (640px - 1024px)
- Sidebar pode colapsar
- Grid de 2 colunas
- Hero section ajustado
- Features em 2 colunas

### Mobile (< 640px)
- Sidebar em drawer/menu hambúrguer
- Grid de 1 coluna
- Hero section vertical
- Features empilhadas
- Botões full-width
